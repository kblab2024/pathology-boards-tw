## Context · 為什麼要做這份

使用者觀察「lung 考得越來越臨床，尤其 115-1」。證據：

| 115-1 題 | 主題 | 測驗手法 | Robbins 對應 |
|---|---|---|---|
| **115-1-90** | Pulmonary hypertension | 找錯的敘述（concept trap） | p.640–642 |
| **115-1-91** | Lobar pneumonia 四期 | 病理分期 + 致病菌 | p.644 + p.34 OCR |
| **115-1-92** | Bronchiectasis | 臨床表現排除（**hyperventilation 不是**） | p.625–626 |

向前回顧吻合：
- **114-2-87** SCLC: TP53 + RB1（Robbins p.654「universal inactivation of both TP53 and RB」）
- **108-2-80** SCC + hypercalcemia: PTHrP / 副贅瘤推理
- **112-2-86** 非吸菸亞洲女性 → adenocarcinoma + EGFR
- **109-1-88** EGFR mutation 臨床決策
- **107-1-89** 女性最常見 = adenocarcinoma（Robbins p.652「now the most common form of lung cancer in adult females and males」）

**整體趨勢**：vignette / pathophysiology / paraneoplastic / demographic / 概念辨誤 → 病理（過去是 buzzword → 病名）

## 1 · 詳細準備方向（七個切入角度，全部 Robbins-anchored）

### 角度 A · 臨床表現 → 病理（反向推理）
- **動機**：115-1 三題都靠這個。
- **訓練目標**：vignette 給症狀組合、demographic、職業史、影像 → 30 秒內鎖定 3 個 ddx。
- **代表組合**：
  - 中年男礦工 + 上葉結節 + eggshell calcification → Silicosis（Robbins p.631–632）
  - 非吸菸亞洲女 + 周邊結節 → Adenocarcinoma EGFR+（Robbins p.652「EGFR ... higher percentage of nonsmoking Asian females」）
  - 老菸槍中央型 + SIADH 或 Cushing 或 Lambert-Eaton → SCLC（Robbins p.657）
  - 鴿主 + 上葉 GGO + BAL CD4/CD8 < 1 → HP（Robbins p.636）
  - 造船工 + 下葉 + pleural plaque + ferruginous body → Asbestos → 警覺 mesothelioma（Robbins p.631 + p.661）

### 角度 B · Paraneoplastic syndrome ↔ 肺癌亞型
- **動機**：108-2-80（高血鈣→鱗癌）、SCLC 系列題反覆出現。
- **Robbins 直接列表（p.657）**：
  - **ADH** → SIADH／低血鈉 → **SCLC**（dominantly）
  - **ACTH** → Cushing syndrome → **SCLC**（dominantly）
  - **PTHrP + PGE + cytokines** → hypercalcemia → **SCC**（mostly）
  - **Calcitonin** → 低血鈣
  - **Gonadotropin** → 男性乳房發育
  - **Serotonin / bradykinin** → carcinoid syndrome
  - **其他**：Lambert-Eaton（anti-VGCC，任何亞型但多 SCLC）、acanthosis nigricans、leukemoid reaction、**Trousseau syndrome**（DVT/血栓栓塞）、**HPOA**（杵狀指）
- **頻率**：「1%–10%」臨床顯著（Robbins 原文）

### 角度 C · 分子地景 → 治療決策（採 Robbins 確切數字）
- **Adenocarcinoma**（Robbins p.652）：
  - **EGFR**：白人 10–15%，**非吸菸亞洲女更高**
  - **ALK**：3–5%
  - **ROS1**：1%
  - **MET**：2–5%
  - **RET**：1–2%
  - **BRAF**：2%
  - **PI3K**：2%
  - **KRAS**：~30%（不要寫 13%；13% 是 G12C 子型）
- **Squamous**（Robbins p.653）：TP53（dysplasia 10–50%、CIS 60–90%）、**CDKN2A/p16** 65%、**FGFR1** 擴增
- **SCLC**（Robbins p.654）：「universal inactivation of both TP53 and RB」、NSCLC→SCLC transformation 透過 RB loss
- **PD-L1 / 免疫療法閾值**：Robbins 提到 TKI 與 KRAS inhibitor 為主，PD-L1 IHC 閾值來自臨床指引（不在本章核心），仍要在表中列出（TPS ≥50% / ≥1% / <1%）以接軌新藥題

### 角度 D · ILD 三足鼎立（UIP vs NSIP vs HP）— Robbins 直接證據
- **UIP/IPF**（Robbins p.627–628）：
  - **空間 + 時間異質性**（spatial AND temporal heterogeneity）
  - **Fibroblastic foci**（早期病灶，藍綠 myxoid）
  - **Honeycomb fibrosis**（晚期）
  - **下葉、胸膜下、interlobular septa**
  - **基因**：TERT / TERC / PARN / RTEL1 telomere maintenance（家族型 15%）、**MUC5B promoter SNP**（散發 1/3）、surfactant gene mutation（罕）
  - 治療：肺移植；TKI（nintedanib）+ TGF-β antagonist（pirfenidone）可緩慢延緩
- **NSIP**（Robbins p.628）：
  - **時間同質性**（temporal homogeneity）— 與 UIP 的最重要鑑別
  - 分 cellular vs fibrosing；cellular 預後較好
  - **無** fibroblastic foci、honeycombing、hyaline membrane、granuloma
  - 常為女性非吸菸、第六十年代、雙側下葉對稱、reticular opacity
  - **可用免疫抑制劑**（與 UIP 不同 — UIP 不應用）
- **HP**（Robbins p.636）：
  - 急性 → 亞急性（centered on bronchioles）→ 慢性
  - **Nonnecrotizing / loosely formed granulomas**（2/3 病人）
  - **Bronchocentric injury**（peribronchiolar metaplasia of alveolar duct/respiratory bronchiole）
  - 間質 lymphocytes + plasma cells + macrophages；**eosinophil rare**（與 eosinophilic pneumonia 鑑別）
  - 慢性可進展至 UIP-pattern
  - BAL：CD4 + CD8 T 都上升

### 角度 E · 感染分期與致病菌排序（115-1-91 證據）
- **Lobar pneumonia 四期**（Robbins p.644 — 直接引用）：
  1. **Congestion**：肺重、boggy、紅；血管 engorgement、肺泡水腫、**少量**中性球、**大量細菌**
  2. **Red hepatization**：massive confluent exudation；中性球 + RBC + fibrin 充滿肺泡；肉眼紅、堅實、無氣、肝樣
  3. **Gray hepatization**：RBC 崩解；**纖維化膿性**滲出持續；灰棕色
  4. **Resolution**：酵素分解 → 顆粒狀半流質碎屑被 macrophage 吞噬、咳出、或纖維母細胞 organize
- **CAP 致病菌**（Robbins p.644–645）：S. pneumoniae（lobar 最常見、rust-colored）→ H. influenzae / M. catarrhalis（COPD 加重）→ S. aureus（病毒感染後續發）→ Klebsiella（酒癮、currant-jelly、上葉空洞）→ Pseudomonas（院內、neutropenic）→ Legionella（水冷塔/熱水、移植病人、**urine antigen 快速診斷**）→ Mycoplasma（年輕、學校／軍營）
- **HIV CD4<200**：PCP（cup-shape silver stain）、CMV（owl-eye）、Cryptococcus（India ink、mucicarmine+）

### 角度 F · 血流動力學與右心連動（115-1-90 證據）
- **Pulmonary HTN 定義**（Robbins p.640）：mean PA ≥25 mmHg at rest
- **WHO 5 groups**（Robbins p.640，原文直引）：
  1. **PAH**（小肺動脈病變；includes idiopathic/heritable、結締組織病、HIV、藥物、左→右分流）
  2. 左心衰所致
  3. 肺部疾病／低氧（COPD、ILD、OSA）
  4. 慢性血栓栓塞（CTEPH）
  5. 不明／多因
- **病理**（Robbins p.641）：「**all forms** have medial hypertrophy + RV hypertrophy」；arterioles 40–300 μm 顯著 medial hypertrophy + intimal fibrosis
- **Plexiform lesion**（Robbins p.641）：「**one extreme in the spectrum**」「**characteristic of advanced** pulmonary hypertension」「most prominent in idiopathic/familial（Group 1）、unrepaired L→R shunts、HIV、drugs」
  - **115-1-90 陷阱**：「plexiform 是 PAH 的**早期**變化」→ 錯。Robbins 寫的是 "advanced" / "extreme"
- **BMPR2 mutation**（Robbins p.640）：家族型 75%、散發 25%（autosomal dominant、incomplete penetrance）
- **Heath-Edwards I–VI 分期注意**：09 weak-spot #12 既有此分期表；本版 Robbins 10th edition **未明列 I–VI**，僅描述為 spectrum + plexiform = extreme。**Phase 2 寫 handbook 時保留 Heath-Edwards 表（仍是經典考點）但加註此分期屬古典文獻，Robbins 強調為「spectrum with plexiform = advanced」**
- **Acute vs chronic cor pulmonale**（105-1-82 考點）：acute = RV dilation **no hypertrophy**；chronic = RV hypertrophy

### 角度 G · 職業／環境暴露 fingerprint（Robbins p.630–632 表 15.6 對照）
- **粒徑 1–5 μm 最危險**（Robbins p.630）
- **Silica**：上葉、whorled collagen nodule、polarizable、**eggshell LN**、TB 風險↑、**progressive massive fibrosis (PMF)**、Caplan syndrome
- **Asbestos**（serpentine chrysotile 90% vs amphiboles 致 mesothelioma 力更強）：下葉、**ferruginous body**、pleural plaque（橫膈面）、asbestosis + 肺癌（與菸**強協同**）+ mesothelioma（少量也可，無菸關）
- **Coal**：下葉、anthracotic、macule、Caplan、PMF
- **Beryllium**：非乾酪化肉芽腫（似 sarcoid）、HLA-DPB1 Glu69 易感

### 表 2-10 · Bronchiectasis 臨床三角（Robbins p.625–626）— 直接回應 115-1-92

## 3 · 「臨床轉向題型 drill」獨立節（重點）

Phase 2 lung-summary.md 中最具差異化的一節。

### 3-A · 115-1 三題完整解析（Robbins-anchored）

每題：
1. 抄題幹（國考代碼合法）
2. 各選項對錯依據引 Robbins 章節（不洩漏 PDF 路徑）
3. 標示「為什麼不是純背 buzzword」
4. 教「下次遇到類似結構怎麼破題」

預定要點：
- **115-1-90 PAH**：plexiform = Robbins p.641「extreme of spectrum / advanced」→ 「早期」一定錯；WHO 5 groups 重新列出；secondary（左心、ILD、OSA）佔大宗
- **115-1-91 Lobar pneumonia**：四期口訣「**充紅灰消**」+ S. pneumoniae 為主因；rust-colored sputum + lobar consolidation
- **115-1-92 Bronchiectasis**：症狀三角（cough + purulent/foul sputum + dyspnea + hemoptysis），**hyperventilation 不是**；強調 obstructive RESP insufficiency

### 3-B · 反向推理練習（5 題，從題庫挑）

格式：vignette → 30 秒鎖 3 ddx → 一個關鍵點切開。

- **112-2-86**：年輕非吸菸亞洲女性 → AdenoCa + 查 EGFR/ALK

- **114-2-87**：TP53 + RB1 universal → SCLC
- **104-2-85**：年輕非吸菸者下葉 panacinar 肺氣腫 → α1-AT def

### 3-C · 「找錯的敘述」題型陷阱清單（concept trap）

- **「Plexiform lesion 是 PAH 的早期變化」→ 錯**（Robbins: extreme/advanced）
- **「Acute cor pulmonale 有 RV 肥厚」→ 錯**（只擴張；chronic 才肥厚）
- **「Asbestos 必須大量暴露才會 mesothelioma」→ 錯**（low-level 也可，amphibole 尤甚）
- **「Silicosis 不會增加 TB 風險」→ 錯**（顯著增加）
- **「α1-AT def 是上葉 centriacinar」→ 錯**（下葉 panacinar）
- **「NSIP 應該用免疫抑制」→ 對；「UIP 應該用免疫抑制」→ 錯**（UIP 反而傷害）
- **「Cryptogenic organizing pneumonia 有 fibroblastic foci」→ 錯**（是 Masson body，間質完整）
- **「Carcinoid 用 Ki67 分級」→ 錯**（肺 carcinoid 用形態，不用 Ki67；GI/胰才用）

## 6 · 已知 Robbins 引用注意事項

1. **Heath-Edwards I–VI 分期**：Robbins 10th edition Ch 15 **未使用此分期**；以 "spectrum, with plexiform lesion as extreme" 描述。`handbook/09` weak-spot #12 既有 I–VI 表（屬古典分期），Phase 2 加註保留以利國考。
2. **BerEp4 = MOC-31** 抗體名稱差異：Robbins 用 BerEp4，業界常用 MOC-31，兩者同抗原；handbook 表 2-3 並列。
3. **KRAS 30% 為全體 KRAS** 變異率；KRAS G12C 子型約佔 13%（治療相關，sotorasib/adagrasib 限 G12C）。表 2-5 已分列。
4. **Plexiform lesion 與 group 歸屬**：Robbins p.641 將 "unrepaired L→R shunt" 列為 group 2 是版本／OCR 疑點，**現行 WHO 共識歸 group 1（PAH associated with congenital heart disease）**。handbook 採現行 WHO，並備註。

## 7 · 對 115-2 的大膽預測（AI 趨勢外推）

> ⚠️ **免責**：以下純屬由 (a) 104-1～115-1 題庫題型移動軌跡 + (b) Robbins 10th ed Ch 15 強調重點 + (c) 國際肺科臨床新進展三者交叉外推。**不是內幕、不是命題委員意圖**，請當作「該章複習優先序」使用，不可當賭注。

### 趨勢前提（用以推論）

1. **115-1 已連續三題走「臨床／概念辨誤」**（90/91/92），且 114-2 開始（114-2-87 SCLC TP53+RB）也走分子推理；題型重心明確從「buzzword → 病名」轉向「vignette / pathophys / 概念 → 病理」。
2. **Robbins 10th 強調但本十年題庫未直接考過**的主題，被「補考」機率最高。
3. **新藥／新分子標記**（osimertinib、sotorasib、PD-L1 IHC）入題機率上升，因 109-1-88 已開先例。
4. 國考傾向「同主題不連續兩屆重考」；115-1 已考 PAH／pneumonia stages／bronchiectasis，**115-2 同主題重出機率較低**。

---

### Tier S · 高機率（≥70% 至少有一題命中）

**S1. Hypersensitivity Pneumonitis（HP）** — 過去十年題庫**從未直接以 HP 為主答案**，但 Robbins p.636 給整節並列三特徵：non-necrotizing/loosely formed granuloma、bronchocentric、淋巴細胞為主（**eosinophil 罕**）。預測 vignette：「鴿主或農夫 + 上中葉 GGO + BAL lymphocytosis + 非乾酪化肉芽腫」→ HP。陷阱選項：sarcoidosis（granuloma 較緊實、bilateral hilar LAD）、UIP（無 granuloma）。

**S2. PD-L1 / 免疫療法閾值** — 109-1-88 已開分子治療先例，但**檢查站抑制劑從未直接入題**。Robbins p.658 提到 KRAS inhibitor + TKI。預測題型：「NSCLC TPS ≥50% / ≥1% / <1% → 一線方案選擇」或「驅動突變陰性 + PD-L1 高表現 → pembrolizumab」。

**S3. KRAS（含 G12C / sotorasib）** — Robbins p.652 直接寫 KRAS 在 NSCLC 占 **~30%**（最常見驅動），且 p.657 KEY CONCEPTS 點名 **KRAS inhibitor**。預測題型：「最常見 NSCLC 驅動基因 = KRAS」（陷阱選 EGFR），或「KRAS G12C → sotorasib」。

**S4. ILD 三方鑑別**（UIP vs NSIP vs HP）— Robbins 直接用整節並列；題庫單獨考過 NSIP（104-1-89）、UIP（107-1-87、111-2-87、114-1-87），**從未三方對照**。預測「temporal homogeneity → NSIP」或「給 cellular pattern → 哪個診斷預後最好」。

**S5. Sarcoidosis** — 過去十年 lung 題庫**直接以 sarcoid 為答案的題目稀少**（多落在免疫／granuloma 章節），但 Robbins p.634–636 給大塊：CD4+ Th1、IFN-γ + IL-2、**Schaumann body / asteroid body**、bilateral hilar LAD、肺侵犯 90%、erythema nodosum。預測 vignette：「青壯非裔／日本人 + 雙側肺門 LAD + 非乾酪化 granuloma + uveitis」→ sarcoid。

---

### Tier A · 中高機率（40–70%）

**A1. Carcinoid（typical vs atypical）vs SCLC** — Robbins p.658 強調 carcinoid 「**單獨分類於 neuroendocrine carcinoma 之外**」「**不用 Ki67 分級**」（與胃腸/胰不同），是新版 Robbins 的明確強調點。預測：「20% 非吸菸 + 中央 polypoid + 整齊器官樣 + scattered Syn+Chr+ + <2 mitoses/2mm² → typical carcinoid」。陷阱：用 Ki67 分級（錯）、SCLC（mitosis 高太多）。

**A2. Mesothelioma 分子**（BAP1、NF2、CDKN2A 80%）— 過去十年只考過 mesothelioma 形態與 calretinin IHC（104-2-87、110-2-87、113-1-87），**分子從未入題**。Robbins p.661 直接列出三個基因；BAP1 germline mutation 高罹病率近年國際強調。預測：「mesothelioma 最常見體細胞變異 = CDKN2A loss（80%）」或「BAP1 germline + 家族 mesothelioma」。

**A3. Goodpasture / DAH** — Robbins p.642。Anti-GBM、type II 過敏、**linear IIF along GBM**、肺（hemoptysis）+ 腎（GN）。題庫 06 高頻清單已列但未直接考。預測：「青年男 + hemoptysis + hematuria + linear IIF」→ Goodpasture。

**A4. GPA（granulomatosis with polyangiitis）肺侵犯** — Robbins p.643。**c-ANCA / PR3-ANCA**、上呼吸道（鼻 saddle nose）+ 肺空洞 + GN。題庫 06 列但未直接考。預測：「鼻潰瘍 + 肺空洞 + c-ANCA」→ GPA。

**A5. Asthma 病理**（Curschmann spiral、Charcot-Leyden crystal、basement membrane 增厚、smooth muscle 肥厚、eosinophil 浸潤）— 過去十年 asthma 病理罕被直接考。Robbins p.622–625 給整節 + Fig 15.11。預測：「痰玻片：螺旋狀膠絲與菱形晶體」→ asthma。

---

### Tier B · 中機率（20–40%）

**B1. CF / 原發性纖毛運動異常 / Kartagener** — 115-1 已考 bronchiectasis 結果；後續可能延伸 cause（CF CFTR、PCD dynein、situs inversus + 鼻竇炎 + 男性不孕）。Robbins p.625。

**B2. Pulmonary alveolar proteinosis（PAP）** — Robbins p.637。**Anti-GM-CSF autoantibody**、PAS+ 肺泡內物質、**crazy-paving** 影像。從未考過，但 Robbins 給整節。

**B3. LAM（lymphangioleiomyomatosis）** — Robbins p.637。年輕女性 + 反覆氣胸 + chylothorax + 囊狀肺 + HMB-45+。從未考過，但屬「漂亮病例」。

**B4. EGFR T790M / C797S 抗藥** — Robbins 提及 TKI 抗藥；osimertinib 解 T790M、C797S 解 osimertinib。屬高階分子題，可能入題。

**B5. ARDS / DAD 分期時序** — 過去十年僅 105-1-87 考過 acute interstitial pneumonia 概念。Robbins p.615–617 給 exudative → proliferative → fibrotic 三期 + 死亡率數字（mild 35% / moderate 40% / severe 46%）。

**B6. 肺移植 chronic rejection / BOS** — 已考過一題（obliterative bronchiolitis）。Robbins p.650。可能換角度重考。

---

### Tier C · 黑馬（<20%，但出了會很痛）

- **Lepidic-only growth = adenocarcinoma in situ**（Robbins p.652 直接定義）
- **IgG4-related lung disease**
- **Pulmonary Langerhans cell histiocytosis**（年輕吸菸者 + 上葉星狀結節 + Langerin+/CD1a+/S100+）
- **Idiopathic pulmonary hemosiderosis**（兒童 DAH 反覆，無腎臟）
- **SARS-CoV-2 ARDS 病理**（Robbins 已納入 p.648 + ACE2 受體）— 後 COVID 時期國考可能補考
- **Caplan syndrome**（RA + coal/silicosis nodule + 中央乾酪性）
- **Solitary fibrous tumor of pleura**（CD34+ STAT6+ keratin−，與 mesothelioma 鑑別）— Robbins p.661
