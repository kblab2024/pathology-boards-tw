# data/ — 原始考題 JSON

> _本目錄存放的是中華民國醫師國考病理科官方公告試題的整理。原始題庫由考選部公佈；JSON 結構化版本由 Claude Code 整理。題目與官方答案皆為公開資訊，僅作教學使用。_

## 檔案

| 檔名 | 範圍 | 題數 |
|------|------|------|
| `questions-104-1_to_114-1.json` | 104 年第 1 次 ~ 114 年第 1 次 | 575 |
| `questions-114-2_to_115-1.json` | 114 年第 2 次、115 年第 1 次 | 50 |

合計 **625 題**。

## Schema

兩個檔都是 JSON 陣列。每個元素結構如下：

```json
{
  "題號": 76,
  "年度": 104,
  "第幾次": 1,
  "question": "活體肝移植捐贈者於部分肝切除後剩下的肝臟變化，下列敘述何者正確？",
  "options": {
    "A": "被切除的肝葉重新長出",
    "B": "剩下的肝臟至少需半年以上才能長大一倍",
    "C": "肝細胞與庫弗氏細胞（Kupffer's cells）及內皮細胞同步複製",
    "D": "肝臟切除後的代償性生長主要是由於肝內幹細胞（intrahepatic stem cells）之生長"
  },
  "answer": "C"
}
```

| 欄位 | 型別 | 說明 |
|------|------|------|
| `題號` | integer | 該次考試中的題目編號（不跨年度唯一） |
| `年度` | integer | 中華民國年（104-115） |
| `第幾次` | integer | 該年第幾次考試（1 或 2） |
| `question` | string | 題幹原文（含繁體中文 + 英文醫學詞彙） |
| `options` | object | A/B/C/D 四個選項 |
| `answer` | string | 正確答案（"A"/"B"/"C"/"D"，極少數送分題為 "#"） |

**唯一識別**：`(年度, 第幾次, 題號)` 的三元組。

## 已知資料品質問題

- `questions-104-1_to_114-1.json` 內有 3 題 `question` 為空字串
- `questions-114-2_to_115-1.json` 內 1 題 `answer` 為 `"#"`（官方送分題）
- 無詳解欄位 — 詳解在 `handbook/03-mock-exam-key.md`，內容為 AI 整理產物

## 使用注意

- 這些檔案是**不可變的**原始資料。新題目請另開新檔（如 `questions-115-2.json`），不要編輯既有檔。
- 程式存取時請用 UTF-8 解碼。
