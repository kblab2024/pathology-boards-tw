## 六、必背口訣總集

1. **"Boot-shaped heart" = Fallot** = **VOPR**: VSD、Overriding aorta、Pulmonary stenosis、RV hypertrophy
2. **MI 1-3 天 = 黃軟**（PMN）；**1-2 週 = 紅褐邊**（granulation）
3. **Aschoff body = Anitschkow cell + giant cell + lymphocyte** in RHD myocardium
4. **Auer rod + APL → t(15;17) PML-RARA → ATRA**
5. **5-FOH 失神經**：Coagulative (心肝腎脾)、Liquefactive (腦)、Caseous (TB)、Fat (胰)、Fibrinoid (血管)
6. **腦疝 5 種，Duret = uncal**（transtentorial）
7. **MSA = oligodendrocyte α-syn**；**PD/DLB = 神經元 Lewy body**
8. **EATL = T 細胞**（不是 B！）；**MALT = B**
9. **Plexiform = 晚期 PAH**；早期是 medial hypertrophy
10. **Renal oncocytoma 粒線體；Chromophobe RCC 細胞膜明顯 (plant-cell)**
11. **HNF1α-HCA = LFABP 失**；**β-catenin HCA = nuclear β-catenin + GS 瀰漫**
12. **HGSC = TP53 + WT1 + p16 + PAX8**
13. **Erythema migrans = Lyme**（**bull's-eye** + 培養出 Borrelia）
14. **Pellagra = Niacin (B3) + 3D**（Dermatitis、Diarrhea、Dementia）
15. **Cushing-independent → 最常 adrenal adenoma**
