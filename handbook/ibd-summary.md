---

## 1. 核心對照表：Crohn disease vs Ulcerative colitis

| 項目 | Crohn disease (CD) | Ulcerative colitis (UC) |
|------|--------------------|--------------------------|
| 好發族群 | 雙峰：15–30、60–70 歲；白人、Ashkenazi Jew | 15–30 歲為主；男女相當 |
| 病灶位置 | **任何**消化道（口腔到肛門），**末端迴腸最常**（terminal ileum 40%） | **僅大腸**；自直腸往近端**連續**延伸 |
| 直腸侵犯 | 常**保留**（rectal sparing） | **100% 受侵**（pancolitis 起點） |
| 病灶分布 | **skip lesion**（跳躍式） | 連續性 |
| 侵犯深度 | **transmural**（全層） | 僅 mucosa + submucosa |
| 大體外觀 | **鵝卵石樣**（cobblestone）、線狀深裂縫潰瘍、creeping fat、串珠樣狹窄 | 廣泛淺潰瘍、**假性息肉**（pseudopolyp）、無正常黏膜跳躍 |
| 顯微鏡 | **非乾酪化肉芽腫**（30–50%）、淋巴聚集、纖維化、神經增生 | crypt abscess、crypt distortion、basal plasmacytosis、無肉芽腫 |
| 廔管／狹窄 | **常見**（enterocutaneous、enterovesical、perianal fistula） | 罕見 |
| Toxic megacolon | 較少 | **可能**（最致命急性併發症） |
| 結腸癌風險 | ↑（與 UC 相當，視侵犯範圍） | ↑↑↑（pancolitis ≥ 8–10 年顯著上升） |
| 血清抗體 | **ASCA +**（anti-Saccharomyces cerevisiae） | **p-ANCA +**（atypical perinuclear） |
| 抽菸影響 | **危險因子**（加重病情） | **保護性**（戒菸後反而易發作） |
| 闌尾切除 | 無影響 | **保護性**（年輕切除者 UC 發病率下降） |
| 內科治療 | 5-ASA、類固醇、azathioprine、anti-TNF（infliximab）、ustekinumab、vedolizumab | 5-ASA（首選輕度）、類固醇、anti-TNF、JAK 抑制劑 |
| 外科治療 | **無法治癒**（切除後易復發於吻合口） | **全大腸切除可治癒**（total proctocolectomy + IPAA） |

## 2. 大體（Gross）特徵速查

| 大體所見 | 強烈指向 |
|----------|----------|
| Cobblestone mucosa（鵝卵石） | **Crohn** |
| Creeping fat（脂肪攀爬） | **Crohn**（mesenteric fat 包繞腸壁） |
| String sign（串珠樣狹窄，影像） | **Crohn**（末端迴腸） |
| Skip lesions | **Crohn** |
| Deep linear / serpiginous ulcers | **Crohn** |
| Fistula、stricture、perianal disease | **Crohn** |
| Pseudopolyps（殘存發炎黏膜島） | **UC** > Crohn |
| 連續、由直腸往近端延伸的紅腫脆弱黏膜 | **UC** |
| Backwash ileitis（末端迴腸輕度發炎，但續發於 pancolitis） | **UC** |
| Lead-pipe colon（影像，結腸袋消失） | **UC**（慢性） |
| Toxic megacolon（橫結腸擴張 > 6 cm） | **UC** > Crohn |

## 3. 顯微鏡（Microscopic）特徵速查

| 顯微所見 | Crohn | UC |
|----------|-------|-----|
| **非乾酪化肉芽腫** | **30–50% 有**（diagnostic） | 無（有則重新評估診斷） |
| 透壁性發炎（transmural） | **是** | 否（限於 mucosa + submucosa） |
| 淋巴聚集（lymphoid aggregates） | **多、跨越肌肉層** | 限於 mucosa |
| Crypt distortion / branching | 較少 | **常見** |
| Crypt abscess | 可見 | **典型** |
| Basal plasmacytosis | 較少 | **典型**（早期診斷重要線索） |
| Paneth cell metaplasia（左側結腸） | 可見 | **慢性化指標** |
| Pyloric metaplasia（迴腸） | **典型** | 罕見 |
| 神經叢增生 / 纖維化 | **典型** | 罕見 |
| 黏膜固有層 eosinophil 浸潤 | 兩者皆可 | 兩者皆可 |

> **重點：** 看到「非乾酪化肉芽腫」+「跳躍式」+「末端迴腸」→ Crohn。
> 看到「直腸起連續發炎」+「basal plasmacytosis」+「crypt distortion」+「無肉芽腫」→ UC。

## 4. 腸外表現（Extraintestinal manifestations）

兩者皆可見，但傾向不同：

| 系統 | 表現 | 較常見於 |
|------|------|----------|
| 關節 | 周邊大關節炎、ankylosing spondylitis、sacroiliitis（HLA-B27 相關） | 兩者皆可 |
| 皮膚 | **Erythema nodosum**（紅斑性結節） | 兩者，CD 稍多 |
| 皮膚 | **Pyoderma gangrenosum**（壞疽性膿皮症） | **UC > CD** |
| 眼 | Uveitis、episcleritis | 兩者 |
| 肝膽 | **Primary sclerosing cholangitis (PSC)** | **UC**（約 5%；PSC 患者中 70% 有 UC） |
| 血液 | 缺鐵性貧血、B12 缺乏（CD 切除末端迴腸者） | 兩者 |
| 腎 | 草酸鈣腎結石（CD 脂肪吸收不良） | **CD** |
| 全身 | 發燒、體重下降、生長遲緩（兒童） | 兩者 |

**口訣：** PSC ↔ UC（並考慮膽道癌風險）；腎結石 ↔ CD（脂肪吸收不良 → 草酸吸收增加）。

---

## 5. 併發症一覽

### Crohn disease

- 廔管（enterocutaneous、enterovesical、enterovaginal、perianal）
- 腸狹窄 → 腸阻塞
- 腹腔內膿瘍
- 吸收不良（末端迴腸切除 → B12 缺乏、膽鹽吸收不良 → 脂瀉）
- **小腸腺癌**（罕見但風險上升，特別是長期發炎段）
- 結腸癌（範圍視結腸侵犯）

### Ulcerative colitis

- **Toxic megacolon**（最致命急性併發症）
- 大量出血
- 穿孔
- 狹窄較少（若有，要懷疑癌變）
- **結腸癌**：pancolitis ≥ 8–10 年風險顯著上升；左側結腸炎風險次之；單純直腸炎風險低
- **PSC + UC** → 結腸癌風險倍增

## 6. 血清學與輔助檢查

| 標記 | Crohn | UC |
|------|-------|-----|
| ASCA（anti-Saccharomyces cerevisiae Ab） | **+ 60–70%** | + < 15% |
| p-ANCA（atypical perinuclear） | + 5–15% | **+ 60–70%** |
| Fecal calprotectin | 升高 | 升高（活動性指標，鑑別 IBS） |
| CRP / ESR | 升高（CD 較明顯） | 升高 |

**口訣：** ASCA（C → CD）、p-ANCA（U → UC）。

---

## 7. Indeterminate colitis（IC）

- 約 5–15% IBD 患者在初次切片時無法分類為 CD 或 UC。
- 通常為**僅大腸侵犯**，但具備 CD 與 UC 重疊特徵。
- 處理：先以 UC 模式治療；長期追蹤後約半數會明朗化為 CD 或 UC。
- 對「擬接受 IPAA 手術」者要特別小心 — 若實為 CD，術後 pouchitis 與廔管率高。

---

## 8. Microscopic colitis（常與 IBD 混淆）

> 影像、內視鏡**外觀正常**，臨床表現**慢性水樣腹瀉**（非血便），需切片診斷。
> 好發中老年女性，與 NSAIDs、PPI、SSRI、自體免疫疾病（celiac、甲狀腺）相關。

| 項目 | Collagenous colitis | Lymphocytic colitis |
|------|---------------------|----------------------|
| 性別 | **女性 > 男性**（10–20 : 1） | 女性 ≈ 男性 |
| 顯微特徵 | **subepithelial collagen band > 10 μm**（trichrome 染色清楚） | intraepithelial lymphocyte **> 20 / 100 上皮細胞** |
| 共通點 | 兩者皆有 lamina propria 慢性發炎、上皮細胞剝離 | 同左 |
| 大腸結構破壞 | **無**（crypt 結構保留，與 IBD 區別） | **無** |
| 治療 | 停 NSAIDs / PPI；budesonide 為首選 | 同 collagenous |

**鑑別要點：** 與 UC 區別 — UC 有 crypt distortion、basal plasmacytosis、潰瘍；microscopic colitis 結構正常。
**與 celiac 區別：** Celiac 主要在小腸 + villous atrophy + IEL 增加。

## 9. IBD-associated dysplasia 與監測

長期 UC（特別是 pancolitis）與廣泛 Crohn colitis 患者，腸癌風險上升，需定期內視鏡監測切片。

### Dysplasia 分類（修訂版 Riddell / SCENIC 共識）

| 分類 | 描述 | 處理 |
|------|------|------|
| Negative for dysplasia | 無不典型 | 例行追蹤 |
| Indefinite for dysplasia | 發炎背景下難判斷 | 治療發炎後重新切片 |
| Low-grade dysplasia (LGD) | 細胞核擁擠、深染但維持極性 | 內視鏡切除（若 visible）；不可切除 → 全大腸切除考慮 |
| High-grade dysplasia (HGD) | 極性喪失、嚴重核異常 | **全大腸切除** |
| **DALM**（dysplasia-associated lesion or mass） | 不規則隆起病灶 + dysplasia | **全大腸切除**（傳統觀念） |
| 散發性 adenoma-like polyp | 邊界清楚、可完整切除、周圍黏膜無 dysplasia | 內視鏡切除 + 加密追蹤 |

**SCENIC 共識**（2015）改採「visible vs invisible」「polypoid vs non-polypoid」分類，傾向以高解析度色素內視鏡（chromoendoscopy）+ 標靶切片取代 random biopsy。

### IBD 相關癌 vs 散發性大腸癌

| 項目 | IBD-associated CRC | Sporadic CRC |
|------|---------------------|--------------|
| 發生路徑 | inflammation → dysplasia → carcinoma | adenoma → carcinoma |
| 多病灶 | **常見** | 較少 |
| 位置 | 較分散，左右側皆可 | 多左側 |
| 分子變化 | **早期 p53 mutation**、晚期 APC | 早期 APC、晚期 p53 |
| 組織學 | mucinous、signet-ring 比例較高 | 一般 adenocarcinoma |
| 預後 | 同分期之下相當 | — |

## 10. 國考高頻 buzzwords / 陷阱題提醒

- 「Non-caseating granuloma + skip lesion + terminal ileum」→ **Crohn**
- 「Pseudopolyp + 直腸連續往上 + 無肉芽腫」→ **UC**
- 「Toxic megacolon」→ **首選 UC**（CD 亦可，較少）
- 「PSC + 慢性血便」→ 想 **UC**
- 「Perianal fistula + creeping fat」→ **Crohn**
- 「年輕女性、水樣腹瀉、內視鏡正常、切片見 subepithelial collagen band」→ **Collagenous colitis**（非 IBD）
- 「UC 患者 pancolitis 10 年後 surveillance 切片：HGD」→ **建議全大腸切除**
- 「Anti-Saccharomyces cerevisiae Ab 陽性」→ **CD**
- 「p-ANCA 陽性 + 慢性血便」→ **UC**
- 「闌尾切除後對 UC 發病有保護性」(易考細節)
- 「抽菸對 UC 是保護、對 CD 是危險」(經典反直覺題)
- 「IPAA (ileal pouch-anal anastomosis) 後 pouchitis 發作」→ 多見於 **UC 術後**；若反覆 + 廔管 → 重新評估是否實為 **CD**

```
─────────────────────────────────────
位置    全消化道(末端迴腸)    僅大腸(直腸起)
分布    跳躍                  連續
深度    全層                  黏膜+黏膜下
肉芽腫  30–50% +              無
廔管    +++                   罕見
偽息肉  少                    +++
血清    ASCA+                 p-ANCA+
抽菸    害                    保護
治癒    無(切除復發)          全大腸切除可
腸外    腎結石                PSC、PG
癌變    ↑                     ↑↑↑(pancolitis≥10yr)
急症    膿瘍/廔管/阻塞        Toxic megacolon
```
