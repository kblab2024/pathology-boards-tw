## 1. 核心對照表：肺癌四亞型（Robbins p. 651–657）

| 項目 | Adenocarcinoma | Squamous cell carcinoma | Small cell carcinoma (SCLC) | Large cell carcinoma |
|---|---|---|---|---|
| 流行病學 | **男女皆最常見**；非吸菸者亦最常見；亞洲女性高 | 老菸槍男性 | 強烈菸害；幾乎不發生於非吸菸者 | 菸害 |
| 位置 | 周邊 | 中央／hilar | 中央 | 周邊 |
| 組織學 buzzword | lepidic（**唯一非侵犯型**）／acinar／papillary／micropapillary／solid／mucinous | keratin pearl、intercellular bridge | small blue cells、**Azzopardi**（DNA 沾染血管壁）、crush artifact | 大細胞、無分化、排除診斷 |
| IHC | **TTF-1+ Napsin A+ CK7+** | **p40+** p63+ CK5/6+ | Syn+ Chr+ CD56+ TTF-1+ Ki67↑↑ | TTF-1− p40− |
| 分子（NSCLC 驅動） | EGFR 10–15%（亞洲非吸菸女更高）、ALK 3–5%、ROS1 1%、MET 2–5%、RET 1–2%、BRAF 2%、PIK3CA 2%、**KRAS ~30%** | TP53（CIS 60–90%）、**CDKN2A/p16 65%**、**FGFR1 amp** | **TP53 + RB1 universal**、MYC | 雜（無單一主要驅動） |
| 副贅瘤 | Trousseau、HPOA | **Hypercalcemia（PTHrP）** | **SIADH、Cushing、Lambert-Eaton** | 少見 |
| 治療方向 | TKI（osimertinib、alectinib …）、KRAS G12C inhibitor（sotorasib）、IO | IO + chemo | etoposide + platinum、PCI | chemo / IO |

> **記憶提示**：SCLC 幾乎全部 TP53 + RB **同時失效**（Robbins p. 654「universal inactivation of both TP53 and RB」）；NSCLC 轉變為 SCLC 的常見機制是後天獲得 RB loss。

## 2. NSCLC 分子地景 → 靶向藥（Robbins p. 652、p. 658 KEY CONCEPTS）

| 變異 | NSCLC 頻率（Robbins p. 652） | 一線靶藥 | 抗藥機制 |
|---|---|---|---|
| EGFR L858R / exon 19 del | 10–15%（亞洲非吸菸女更高） | Osimertinib | T790M（解：osimertinib）、C797S |
| ALK rearrangement | 3–5% | Alectinib | G1202R |
| ROS1 fusion | ~1% | Crizotinib／Entrectinib | — |
| MET exon14 skipping | 2–5% | Capmatinib／Tepotinib | — |
| RET fusion | 1–2% | Selpercatinib | — |
| BRAF V600E | ~2% | Dabrafenib + Trametinib | — |
| HER2 ex20 ins | ~2% | Trastuzumab deruxtecan (T-DXd) | — |
| **KRAS（全體）** | **~30%（最常見驅動）** | — | — |
| KRAS G12C 子型 | ~13%（KRAS 中約 4 成） | **Sotorasib／Adagrasib**（限 G12C） | — |
| PIK3CA | ~2% | （NSCLC 暫無核可） | — |

**PD-L1 IHC（免疫療法分流，臨床指引非 Robbins 本章）**：

- TPS ≥ 50% → pembrolizumab 單藥可一線
- TPS 1–49% → 多用 combo（IO + chemo）
- TPS < 1% → chemo ± IO

> **115-2 押題提示**：KRAS 是 NSCLC 最常見驅動（不是 EGFR），是「找最常見」題的常見陷阱選項。

## 3. ILD 三方鑑別：UIP（IPF）vs NSIP vs HP（Robbins p. 627–636）

| 項目 | UIP / IPF | NSIP | HP（chronic） |
|---|---|---|---|
| 時間異質性（temporal） | **有**（早晚共存） | **無**（temporal homogeneity）  | 無 |
| 空間異質性（spatial） | **有**（patchy） | 均勻 | bronchocentric |
| Fibroblastic foci | **典型** | **無** | 罕（chronic 可 UIP-like） |
| Honeycombing | **晚期典型** | 無 | 慢性可有 |
| 肉芽腫 | 無 | 無 | **Non-necrotizing／loosely formed**（bronchocentric，2/3 病人） |
| 淋巴細胞 | 少 | 多（cellular > fibrosing） | 多 |
| Eosinophil | 少 | 少 | **罕**（鑑別 eosinophilic pneumonia） |
| 葉分布 | 下葉、胸膜下、interlobular septa | 下葉對稱 | 上中葉、centrilobular |
| 暴露 | 菸、職業（farming、hairdressing、stone polishing） | 結締組織病／特發 | 鳥糞、moldy hay、加濕器 |
| 基因 | **TERT / TERC / PARN / RTEL1 telomere**（家族 15%）、**MUC5B promoter SNP**（散發 1/3）、surfactant gene 突變（罕） | 多伴 CTD | HLA 相關 |
| 治療反應 | 免疫抑制**無效或有害**；nintedanib + pirfenidone 延緩 | 免疫抑制**可能有效** | 移除暴露可逆 |
| 預後 | 中位 ~3.8 年（Robbins p. 628） | Cellular > fibrosing | 早期可逆 |

> **重點**：UIP 看到「**時間 + 空間皆異質**」+ fibroblastic foci + honeycombing；NSIP 看到「**時間同質**」+ uniform inflammation；HP 看到「**loosely formed granuloma + bronchocentric + lymphocyte 多／eosinophil 罕**」。
> **陷阱**：給 UIP 用類固醇 → 加速惡化；給 NSIP 用 → 可能改善。

## 5. 副贅瘤症候群 ↔ 肺癌亞型（Robbins p. 657）

| 副贅瘤現象 | 機制 | 強烈指向 |
|---|---|---|
| Hypercalcemia | PTHrP + PGE + cytokines | **Squamous**（mostly） |
| SIADH（低血鈉） | 異位 ADH | **SCLC**（dominantly） |
| Cushing | 異位 ACTH | **SCLC**（dominantly） |
| Lambert-Eaton（近端肌無力） | anti-VGCC | **SCLC**（多）；任何亞型可能 |
| Trousseau syndrome（遊走性血栓） | 黏液腫瘤 procoagulant | **Adenocarcinoma**（也見胰、胃） |
| HPOA（杵狀指 + 骨膜炎） | 不明 | **Adenocarcinoma > others** |
| Acanthosis nigricans | — | 任何肺癌 |
| Leukemoid reaction | G-CSF 等 | 任何肺癌 |
| 低血鈣 | 異位 calcitonin | 不指向特定亞型 |
| Gynecomastia | 異位 gonadotropin | 不指向特定亞型 |
| Carcinoid syndrome（潮紅、腹瀉） | Serotonin、bradykinin | **Carcinoid**（肺 carcinoid 較少見） |

> 臨床顯著副贅瘤發生率 **1–10%**（Robbins p. 657）。
> **口訣**：「SC**LC** 給三 C」— SIADH、Cushing、Calcium-channel ab（Lambert-Eaton）；squamous 給高 Ca；adeno 給血栓 + 杵狀指。

## 6. 肺炎致病菌與分期（Robbins p. 644–646）

### 6-1 · Lobar pneumonia 四期（直接回應 115-1-91）

| 期 | 巨觀 | 顯微 |
|---|---|---|
| **Congestion**（1–2 d） | 重、boggy、紅 | 血管 engorgement、肺泡水腫、**少量**中性球、**大量細菌** |
| **Red hepatization**（2–3 d） | **紅、堅實、無氣、肝樣**（hepatization 詞源） | 中性球 + RBC + fibrin 充滿肺泡 |
| **Gray hepatization**（4–7 d） | 灰棕色 | RBC 崩解；**纖維化膿性**滲出持續 |
| **Resolution**（>7 d） | 恢復 | 酵素分解 → 顆粒狀半流質碎屑被 macrophage 吞噬、咳出或纖維母細胞 organize |

> **口訣**：「**充紅灰消**」— 充血 → 紅肝化 → 灰肝化 → 消退。

### 6-2 · CAP 致病菌排序

| 情境 | 首選病原 | 病理／buzzword |
|---|---|---|
| CAP 成人健康 | **S. pneumoniae** | lobar 最常、rust-colored sputum |
| CAP COPD 加重 | H. influenzae、M. catarrhalis | bronchopneumonia |
| 病毒後續發肺炎 | **S. aureus** | 多發空洞、necrotizing |
| 酒癮、糖尿病、慢病 | Klebsiella | **currant-jelly sputum、上葉空洞** |
| 院內 / 中性球低下 | Pseudomonas aeruginosa | necrotizing、bacteremia |
| 旅館空調 / 老人 / 移植 | Legionella | **PCR 或 urine antigen 快速診斷**、低血鈉、腹瀉 |
| Atypical 年輕人 / 學校 / 軍營 | Mycoplasma | 冷凝集、interstitial |
| HIV CD4 < 200 | PCP（Pneumocystis jirovecii） | **silver stain cup-shape、ground-glass** |
| HIV 廣泛 / 移植 | CMV | **owl-eye 包涵體** |
| 鴿糞、AIDS | Cryptococcus | India ink、**mucicarmine+** |

> **115-2 押題提示**：Klebsiella 的「currant-jelly + 上葉空洞」與酒癮聯動，是經典 vignette；Legionella 的「水冷塔 + 低血鈉」近年國際題庫越來越愛考。

## 7. 職業／環境暴露 fingerprint（Robbins p. 630–632）

| 暴露 | 分布 | 病理特徵 | 併發 |
|---|---|---|---|
| Silica（quartz、矽塵） | **上葉** | whorled collagen nodule、polarizable、**eggshell LN calcification** | **TB 風險顯著↑**、PMF、Caplan（合併 RA）、autoimmune |
| Asbestos（amphibole > chrysotile 致 meso） | **下葉** | **Ferruginous body**（含鐵蛋白包覆）、pleural plaque（橫膈面） | **Mesothelioma**（少量暴露即可，無菸關）、肺癌（與菸協同 5×↑）、asbestosis、Caplan |
| Coal | 下葉 | macule、anthracotic pigment | PMF、Caplan |
| Beryllium | 散在 | **非乾酪化肉芽腫**（似 sarcoid）、BeLPT 確診 | HLA-DPB1 Glu69 易感、肺癌（？） |

> **粒徑 1–5 μm 最危險**（Robbins p. 630）— 能到末端小氣道與肺泡，太大被纖毛排出，太小被呼出。
> **陷阱題**：「Asbestos 必須大量暴露才致 mesothelioma」→ 錯（低劑量也可、amphibole 尤甚）；「Silicosis 不增加 TB 風險」→ 錯（顯著增加）。

## 8. PAH 病理學與 WHO 分類（Robbins p. 640–642，回應 115-1-90）

### 8-1 · 全形態共通變化

- **Medial hypertrophy + RV hypertrophy**（all forms 都有，Robbins 原文）
- 小肺動脈（40–300 μm）：medial hypertrophy + **intimal fibrosis**
- **Plexiform lesion = 病程晚期／spectrum 的極端**（Robbins p. 641「one extreme in the spectrum」「characteristic of advanced pulmonary hypertension」）
- BMPR2 mutation：家族型 75%、散發 25%（autosomal dominant、incomplete penetrance）

### 8-2 · WHO 五大分類

| Group | 機制 | 代表病因 |
|---|---|---|
| **1** | **PAH**（小肺動脈本身病變） | 特發、家族（BMPR2）、結締組織病（systemic sclerosis）、HIV、藥物（食慾抑制劑）、congenital L→R 分流 |
| **2** | 左心衰所致 | LV 失能、瓣膜病 |
| **3** | 肺病／低氧 | COPD、ILD、**OSA** |
| **4** | CTEPH | 反覆 PE |
| **5** | 多因／不明 | 血液病、代謝、其他 |

### 8-3 · 古典 Heath–Edwards I–VI 分期（與章 09 #12 接軌）

章 09 弱點補強卡 #12 採古典 Heath–Edwards 六分期（I medial hypertrophy → IV plexiform → VI fibrinoid necrosis），仍是本地考試經典題型。
本版 Robbins 10th ed **未明列 I–VI**，而以「spectrum + plexiform = advanced」描述；兩套說法**結論一致**（plexiform 都是晚期），但用語不同。考試遇到「plexiform 是早期」一律選錯。

### 8-4 · Acute vs Chronic Cor pulmonale（105-1-82 經典題）

| | Acute（如大型 PE） | Chronic（如 COPD 末期） |
|---|---|---|
| RV 變化 | **擴張，無肥厚** | **肥厚** |
| 機轉 | 突發後負荷↑，無時間 remodel | 長期後負荷↑ → 心肌代償肥厚 |

## 9. ARDS / DAD 時序（Robbins p. 615–617）

| 期 | 時間 | 病理 | 臨床 |
|---|---|---|---|
| **Exudative**（acute） | 0–7 d | **Hyaline membrane**（fibrin-rich edema + 壞死上皮殘片）、肺泡水腫、中性球、type I 壞死 | 急性低氧、PaO₂/FiO₂ ↓、refractory hypoxemia |
| **Proliferative / Organizing** | 7–21 d | **Type II pneumocyte 增生**、纖維母細胞、granulation tissue | 部分病人開始恢復 |
| **Fibrotic** | > 3 wk | 間質纖維化、囊狀變化 | 需長期呼吸器、預後差 |

**死亡率**（Robbins p. 615）：mild 35%、moderate 40%、severe 46%；多死於 sepsis／多器官衰竭。

**ARDS 觸發**（Robbins Table 15.2）：**敗血症、瀰漫性肺感染、胃內容物誤吸、機械傷害**佔 50% 以上。

## 10. Bronchiectasis 臨床三角（Robbins p. 625–626，直接回應 115-1-92）

### 10-1 · 症狀（會考的）

- **嚴重持續咳嗽**
- **大量化膿（mucopurulent）、惡臭、帶血絲痰**
- 嚴重時 **dyspnea + orthopnea**
- 偶有 **massive hemoptysis**
- 早晨體位變化時咳嗽加劇（夜間積痰晨起傾倒）

### 10-2 · 「不是」的症狀（115-1-92 陷阱）

- **Hyperventilation 不是 bronchiectasis 的特徵**
- 取而代之的是 obstructive respiratory insufficiency、cyanosis

### 10-3 · 病因分布

- **~50% idiopathic**（Robbins）
- 先天：CF、PCD／Kartagener、intralobar sequestration、免疫缺陷
- 後天感染：嚴重壞死性肺炎、結核
- 阻塞：腫瘤、異物、黏液
- 免疫病：RA、SLE、IBD、post-transplant（chronic rejection、GVHD）
- 過敏：ABPA（asthma 或 CF 患者 + Aspergillus）

### 10-4 · 痰培養（高頻數字）

- **H. influenzae ~50%**
- **Pseudomonas aeruginosa 12–30%**
- 其餘為非結核分枝桿菌等

### 10-5 · 併發

Cor pulmonale、腦膿瘍、amyloidosis（現代抗生素時代減少）。

## 12. 臨床轉向題型 Drill（115-1 同型練習）

### 12-A · 115-1 三題完整解析

**【115-1-90】Pulmonary hypertension — 找錯的敘述**

- 關鍵知識：plexiform lesion 是 **advanced / extreme of spectrum**（Robbins p. 641）；
  「plexiform = 早期變化」是錯誤敘述。
- 鞏固點：mPAP ≥ 25 mmHg 為定義；WHO 5 groups；secondary（left heart、ILD、OSA、CTEPH）佔大宗，純 idiopathic 反而少；BMPR2 75% 家族、25% 散發。
- 破題思路：concept trap 題優先掃過分期／時序／所有形容詞（「早期 vs 晚期」、「肥厚 vs 擴張」、「乾酪 vs 非乾酪」、「上葉 vs 下葉」）。

**【115-1-91】Lobar pneumonia 四期 + 致病菌**

- 關鍵知識：「**充紅灰消**」四期，S. pneumoniae 是 CAP lobar 最常見原因（Robbins p. 644）。
- 鞏固點：red hepatization = neutrophil + RBC + fibrin；gray = RBC 崩解、fibrinosuppurative 持續；resolution 靠 macrophage + fibroblast 清理。
- 破題思路：先把四期由急到慢排列；致病菌依「健康／COPD 加重／酒癮／院內／旅館／HIV」分桶。

**【115-1-92】Bronchiectasis 臨床表現**

- 關鍵知識：cough + purulent/foul sputum + dyspnea + hemoptysis；**hyperventilation 不是**特徵（Robbins p. 626）。
- 鞏固點：原因 50% 特發；CF／PCD／Kartagener／ABPA／obstruction／免疫病；痰培養 H. influenzae 50%、Pseudomonas 12–30%。
- 破題思路：問「以下何者**不是**症狀」時，把症狀分成「常見三角（cough/sputum/dyspnea）」與「偶發（hemoptysis）」，剩下的就是干擾項。

### 12-B · 反向推理 5 題（demographic／paraneoplastic → 病理）

| 題 | 線索 | 一步推理 | 答案 |
|---|---|---|---|
| **108-2-80** | 老菸槍 + 中央型 + **高血鈣** | PTHrP 副贅瘤 | **Squamous cell carcinoma** |
| **112-2-86** | 年輕非吸菸亞洲女性 + 周邊結節 | EGFR/ALK 高機率 | **Adenocarcinoma**（後續查分子） |
| **109-1-88** | 已確診 NSCLC 找到 EGFR L858R | EGFR-TKI 一線 | **Osimertinib**（T790M 也涵蓋） |
| **114-2-87** | Small blue cell tumor + **TP53 + RB1 universal loss** | 神經內分泌癌極端型 | **SCLC** |
| **104-2-85** | 年輕非吸菸者 + 下葉 panacinar 肺氣腫 + 肝硬化 | 缺乏蛋白酶抑制劑 | **α1-antitrypsin deficiency** |

### 12-C · 「找錯的敘述」陷阱清單（concept trap）

下面這些敘述都是**錯的**，遇到就選它：

- 「Plexiform lesion 是 PAH 的早期變化」 → **錯**（Robbins: extreme/advanced）
- 「Acute cor pulmonale 有 RV 肥厚」 → **錯**（只擴張；chronic 才肥厚）
- 「Asbestos 必須大量暴露才會 mesothelioma」 → **錯**（低劑量也可，amphibole 尤甚）
- 「Silicosis 不會增加 TB 風險」 → **錯**（顯著增加）
- 「α1-AT def 是上葉 centriacinar 肺氣腫」 → **錯**（下葉 panacinar）
- 「UIP 應該用免疫抑制劑」 → **錯**（無效或有害；NSIP 才可能有效）
- 「Cryptogenic organizing pneumonia 有 fibroblastic foci」 → **錯**（是 Masson body，間質完整）
- 「肺 Carcinoid 用 Ki67 分級」 → **錯**（用形態；GI／胰才用 Ki67）
- 「Lepidic pattern 是侵犯型 adenocarcinoma」 → **錯**（**唯一的非侵犯型**生長型）
- 「Bronchiectasis 會出現 hyperventilation」 → **錯**（出現 obstructive insufficiency）

### 12-D · Demographic / 暴露 → 最像診斷 速查決策樹

```
非吸菸亞洲女性 + 周邊結節 → AdenoCa（查 EGFR / ALK）
老菸槍 + 中央 + 高血鈣 → Squamous（PTHrP）
```

## 13. 對 115-2 的大膽預測（AI 趨勢外推）

> ⚠️ **免責**：以下純屬由 (a) 104-1 到 115-1 題庫題型移動軌跡 + (b) Robbins 10th ed Ch 15 強調重點 + (c) 國際肺科臨床新進展三者交叉外推。**不是內幕、不是命題委員意圖**，請當作「該章複習優先序」使用，不可當賭注。

### 13-A · 趨勢前提

1. 115-1 已連續三題走「臨床／概念辨誤」（90/91/92），114-2-87 也走分子推理；題型重心從「buzzword → 病名」轉向「vignette／pathophys／概念 → 病理」。
2. Robbins 10th 強調但本十年題庫未直接考過的主題，被「補考」機率最高。
3. 新藥／新分子標記（osimertinib、sotorasib、PD-L1 IHC）入題機率上升，因 109-1-88 已開先例。
4. 國考傾向「同主題不連續兩屆重考」；115-1 已考 PAH／pneumonia stages／bronchiectasis，**115-2 同主題重出機率較低**。

### 13-B · Tier S（高機率，≥ 70%）

| # | 預測主題 | 為什麼 | 預測題型 |
|---|---|---|---|
| S1 | **Hypersensitivity pneumonitis** | 過去十年題庫從未直接考；Robbins p. 636 整節描述三特徵 | 「鴿主 + 上中葉 GGO + 非乾酪化肉芽腫 + BAL lymphocytosis」→ HP |
| S2 | **PD-L1 / 免疫療法閾值** | 109-1-88 開分子治療先例，IO 從未直接考 | 「NSCLC TPS ≥50% → 一線方案 = pembrolizumab 單藥」 |
| S3 | **KRAS / G12C / sotorasib** | Robbins p. 652 直寫 KRAS ~30%（最常見驅動）；p. 658 點名 KRAS inhibitor | 「最常見 NSCLC 驅動 = KRAS」（陷阱 EGFR）；「G12C → sotorasib」 |
| S4 | **ILD 三方鑑別** | Robbins 並列三節；題庫單獨考過 NSIP／UIP，從未三方對照 | 「temporal homogeneity → NSIP」 |
| S5 | **Sarcoidosis** | 過去十年 lung 直接以 sarcoid 為答案的題目稀少；Robbins p. 634–636 大塊 | 「雙側肺門 LN + 非乾酪化肉芽腫 + uveitis」→ sarcoid |

### 13-C · Tier A（中高機率，40–70%）

| # | 預測主題 | 為什麼 |
|---|---|---|
| A1 | **Carcinoid（typical vs atypical）vs SCLC** | Robbins p. 658 強調 carcinoid **不用 Ki67 分級**（與 GI／胰不同） |
| A2 | **Mesothelioma 分子**（BAP1、NF2、CDKN2A 80%） | 過去十年只考形態與 calretinin，分子從未入題 |
| A3 | **Goodpasture / DAH** | Anti-GBM、linear IIF；題庫高頻清單列但未直接考 |
| A4 | **GPA 肺侵犯** | c-ANCA / PR3-ANCA + 鼻 saddle + 肺空洞 + GN，未直接考 |
| A5 | **Asthma 病理** | Curschmann spiral、Charcot-Leyden crystal；過去十年罕被直接考 |

### 13-D · Tier B（中機率，20–40%）

- **CF / 原發性纖毛運動異常 / Kartagener**（115-1 已考 bronchiectasis 結果，cause 可能延伸）
- **Pulmonary alveolar proteinosis (PAP)** — anti-GM-CSF autoantibody、PAS+、crazy-paving；從未考過
- **LAM** — 年輕女性 + 反覆氣胸 + chylothorax + HMB-45+；從未考過但屬經典 case
- **EGFR T790M / C797S 抗藥** — osimertinib 解 T790M、C797S 解 osimertinib
- **ARDS / DAD 分期時序** — 過去十年僅 105-1-87 考過 acute interstitial pneumonia 概念
- **肺移植 chronic rejection / BOS** — 已考過一題，可能換角度重考

### 13-E · Tier C（黑馬，< 20%）

- **Lepidic-only growth = adenocarcinoma in situ**（Robbins p. 652）
- **IgG4-related lung disease**
- **Pulmonary Langerhans cell histiocytosis**（年輕吸菸者 + 上葉星狀結節 + Langerin+/CD1a+/S100+）
- **Idiopathic pulmonary hemosiderosis**（兒童 DAH 反覆，無腎臟）
- **SARS-CoV-2 ARDS 病理**（Robbins 已納入；ACE2 受體）— 後 COVID 時期可能補考
- **Caplan syndrome**（RA + coal／silicosis nodule + 中央乾酪性）
- **Solitary fibrous tumor of pleura**（CD34+ STAT6+ keratin−，與 mesothelioma 鑑別）

### 13-F · 題型結構預測

| 結構 | 出現機率 |
|---|---|
| 「找錯的敘述」概念辨誤 | **≥ 1 題** |
| Vignette → 診斷 | **≥ 1 題** |
| 分子 + 治療連動 | **≥ 1 題** |
| IHC 面板鑑別 | **≥ 1 題** |
| 純形態 buzzword | 仍有但占比下降 |

**總題數預測**：3–5 題，貼近近五年平均。

### 13-G · 必背高 yield 數字 / buzzword（115-2 押題）

- **mPAP ≥ 25 mmHg** = PAH 定義（Robbins p. 640）
- **BMPR2 mutation** 家族 PAH 75%、散發 25%（Robbins p. 640）
- **粒徑 1–5 μm** 最危險（Robbins p. 630）
- **MUC5B promoter SNP** 約 1/3 散發 IPF（Robbins p. 627）
- **CDKN2A loss** mesothelioma 80%（Robbins p. 661）
- **EGFR 亞洲非吸菸女性 > 西方 10–15%**（Robbins p. 652）
- **KRAS ~30%** NSCLC 最常見驅動（Robbins p. 652）
- **TP53 + RB1** SCLC universal（Robbins p. 654）
- **CD4/CD8 < 1** in BAL → HP（教科書 + 臨床指引）
- **Lobar pneumonia 四期**：congestion → red → gray → resolution（Robbins p. 644）
- **HP granuloma**：non-necrotizing／loosely formed／bronchocentric（Robbins p. 636）
- **Lepidic** = AdenoCa **唯一**非侵犯型（Robbins p. 652）
