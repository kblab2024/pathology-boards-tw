## Part 1 — 114 年第 2 次詳解

### Q1（答 D）coagulative necrosis 顯微特徵
- **核心知識**：凝固性壞死 = 細胞質強烈嗜酸性紅染、核淡化或消失（karyolysis），但細胞結構輪廓 ghost-like 仍可辨認
- **干擾選項拆解**：
  - (A) 液化性壞死 → 腦梗塞、化膿，整片融解
  - (B) 乾酪性壞死 → 結核中央，似 cheese 碎屑，無細胞結構
  - (C) 纖維素樣壞死 → 血管壁，PAS+ 強嗜酸帶
- **教學延伸**：心肌梗塞為典型 coagulative；脾、腎、心、肝 ischemic infarct 皆為 coagulative

### Q2（答 A）不可逆細胞傷害標誌
- **核心知識**：粒線體膜通透性轉換孔（mPTP）開啟 → 粒線體蛋白（cytochrome c）外漏 = **不可逆 point of no return**
- **干擾選項**：(B)(C)(D) 都是「可逆」傷害的早期變化
- **教學延伸**：可逆 vs 不可逆的三大分水嶺：① 嚴重粒線體破壞 ② 細胞膜廣泛缺損 ③ DNA 與核蛋白溶解
- **臨床應用**：troponin 升高 = 心肌已進入不可逆，因為它是粒線體釋出物

### Q3（答 D）Marfan syndrome
- **核心知識**：**Fibrillin-1（FBN1）** 突變，**體染色體顯性**遺傳，**身材高瘦**（非矮小），主動脈根部擴大 → 主動脈剝離
- **錯誤選項解構**：
  - (A) 不是 collagen → 是 **fibrillin-1**（彈性蛋白支架）
  - (B) 不是隱性 → 是顯性
  - (C) 不是矮小 → 是高瘦、長指
- **重要對比**：Ehlers-Danlos = collagen 異常，Osteogenesis imperfecta = collagen I 異常
- **教學重點**：「主動脈剝離 + 高瘦」=Marfan

### Q4（答 B）Pemphigus vulgaris
- **核心知識**：抗 desmoglein 3/1 IgG → **Type II 過敏反應**（抗體對抗細胞表面抗原）
- **DIF 影像**：表皮內**網狀魚網樣**（fish-net）沉積
- **臨床特徵**：脆弱性鬆弛大水皰、Nikolsky sign 陽性
- **對比**：Bullous pemphigoid → anti-BP180/230，基底膜帶**線狀**沉積，仍屬 Type II 但水皰較紮實
- **治療**：類固醇 + **Rituximab**（抗 CD20）

### Q5（答 C）慢性 CO 中毒
- **核心知識**：慢性低劑量 CO → **basal ganglia**（特別是 globus pallidus）對缺氧最敏感 → 雙側對稱性壞死
- **急性高劑量 CO**：cherry red 皮膚顏色 + 廣泛大腦皮質損傷
- **延伸**：缺氧最易壞死的部位序列 = 海馬 CA1 (pyramidal) > Purkinje 細胞 > 大腦第 3、5、6 層皮質 > basal ganglia

### Q6（答 A）惡性腫瘤最不可靠特徵
- **核心知識**：**生長快速**不可靠 — 良性也可能快速生長（如妊娠中乳腺纖維腺瘤）；惡性也可能緩慢
- **最可靠 = (D) 轉移**（pathognomonic for malignancy）
- **教學重點**：四大可靠惡性特徵 = ① 轉移 ② 侵犯 ③ 嚴重 atypia + 異常 mitosis ④ 失分化

### Q7（答 A）先天心臟病與肺高壓
- **核心知識**：**左向右分流**（VSD、ASD、PDA）→ 肺血流增加 → 久了肺血管阻塞性病變 → 肺高壓 → **Eisenmenger** 反轉
- **干擾選項**：
  - (B) Fallot 是右向左 → 肺血流**減少**，無肺高壓
  - (C) Coarctation → 上肢高血壓不是肺高壓
  - (D) Isolated PS → 右心室代償，肺壓不升
- **教學延伸**：肺血管病變六階段（Heath-Edwards classification）

### Q8（答 C）非血小板缺陷的凝血異常
- **核心知識**：
  - (A) vWD → vWF 缺，影響血小板黏附（**也**算血小板功能）
  - (B) 尿毒症 → 血小板功能異常
  - (D) Aspirin → 抑制 COX → 血小板功能異常
  - **(C) Hemophilia = Factor VIII/IX 缺失（凝固因子，非血小板）**
- **教學延伸**：BT vs aPTT 鑑別出血傾向
  - BT 延長 = 血小板異常或 vWD
  - aPTT 延長 = 內生路徑（Hemophilia）
  - PT 延長 = 外生路徑（VII 缺、warfarin）

### Q9（答 C）Hodgkin lymphoma 腫瘤細胞稀少
- **核心知識**：Hodgkin lymphoma 中真正的腫瘤細胞 **Reed-Sternberg 細胞**只佔少數（< 5%），其餘為反應性發炎細胞（lymphocyte、eosinophil、plasma cell、neutrophil）
- **RS 細胞**：CD15+、CD30+、CD45−、PAX5 弱+，雙葉核「貓頭鷹眼」
- **對比**：DLBCL、Burkitt、ALCL 腫瘤細胞 = 主體

### Q10（答 D）骨髓造血細胞最少
- **核心知識**：**Aplastic anemia** = 骨髓**空蕩**、被脂肪取代
- **其他三者**骨髓 cellular ↑：
  - IDA → erythroid hyperplasia
  - Megaloblastic → 巨母紅血球性 hyperplasia
  - Thalassemia → 嚴重 erythroid hyperplasia（骨髓擴張 → crew-cut skull）

### Q11（答 A，官方送分）鼻咽癌
- **學理正確答案 = A**：鼻咽癌與 **EBV** 相關，非 HPV
- 但因爭議性（南方型 keratinizing 偶爾與 HPV 相關），官方判送分
- **教學重點**：鼻咽癌「三大特徵」= ① EBV ② 南亞/南中國高發 ③ 頸部淋巴結轉移先表現
- **染色**：EBER ISH 是診斷標配

### Q12（答 B）小細胞肺癌錯誤敘述
- **核心知識**：SCLC 幾乎不帶 EGFR 突變（EGFR 是**腺癌**亞洲女性非吸菸者的特徵）
- SCLC 分子特徵 = **TP53 + RB** 雙失活（>90%）
- SCLC 副贅瘤：SIADH、Cushing（ACTH 分泌）、Lambert-Eaton

### Q13（答 C）矽肺病錯誤敘述
- **核心知識**：矽肺病肺纖維化**不可逆**，停止暴露後**不會恢復**，嚴重者需肺移植
- 矽肺三大特徵：① 上肺葉 nodule ② whorled collagen ③ 偏光下 birefringent silica 粒
- **重要併發症**：矽肺 → 結核**易感**（silicotuberculosis）

### Q14（答 C）口腔癌錯誤敘述
- **核心知識**：口腔 SCC 雖有 TP53/CDKN2A/PIK3CA 突變，**但目前並無有效靶向藥**（這些都是 tumor suppressor、難以針對）
- 治療仍以手術 + 放化療為主，免疫療法（pembrolizumab）為近年新增
- **教學重點**：頭頸 SCC 與肺腺癌「分子相似但治療路徑不同」

### Q15（答 D）HNF1α-inactivated HCA
- **核心知識**：HNF1α 失活 → **LFABP 表現喪失**（題目反過來說 = 錯）
- HCA 四大分子分型（重要！）：
  | 分型 | 標記 | 風險 |
  |---|---|---|
  | HNF1α inactivated | LFABP **失** | 極低惡變 |
  | Inflammatory | SAA、CRP+ | 中等惡變 |
  | β-catenin activated | nuclear β-catenin、GS 瀰漫+ | **高**惡變 |
  | Unclassified | — | 不定 |
- **臨床**：MODY-3 患者多型；好發於女性（口服避孕藥相關）

### Q16（答 B）急性膽囊炎錯誤敘述
- **正確機轉**：阻塞 → 黏膜傷害 → 釋出**前列腺素** → 發炎（題目說「減少」是錯的）
- 90% 結石性，10% 非結石性（與敗血症、創傷、TPN、缺血有關）
- **教學延伸**：膽石症 5F = Female、Fat、Forty、Fertile、Fair

### Q17（答 A）Renal oncocytoma
- **核心知識**：胞質充滿**粒線體** → 嗜酸性
- **形態**：mahogany brown 顏色（肉眼）、central scar（影像）、neoplasm cell 巢狀
- **重要鑑別**：vs Chromophobe RCC（也嗜酸）
  - Oncocytoma 是**良性**
  - Chromophobe RCC 含 microvesicle（PAS+），plant-cell-like 細胞膜

### Q18（答 C）Xanthogranulomatous pyelonephritis
- **核心知識**：**Proteus mirabilis**（urea splitting → struvite stone → 反覆感染）
- 形態：黃色泡沫巨噬細胞（lipid-laden macrophages）瀰漫
- 臨床：女性多、結石、糖尿病
- **腫瘤偽裝**：常被誤以為 RCC（術前難鑑別）

### Q19（答 A）腎細胞癌最大危險因子
- **核心知識**：**抽菸**是首位（其次 HTN、肥胖、慢性腎病、VHL）
- 石綿 → mesothelioma、肺癌（非 RCC）
- 教學延伸：clear cell RCC 與 **VHL** 基因 3p25 缺失相關

### Q20（答 A）放射線治療後乳房肉瘤
- **核心知識**：放射後最常見的二次惡性 = **angiosarcoma**（潛伏期 5-10 年）
- 表現：紫紅斑塊、結節
- **IHC**：CD31、CD34、ERG（內皮標記）+

### Q21（答 A）乳房 ALH 風險
- **核心知識**：Atypical lobular hyperplasia → 雙側乳癌風險 **× 4-5 倍**
- 比 sclerosing adenosis、complex sclerosing lesion 高
- **臨床處置**：診斷後密切追蹤 + 考慮 tamoxifen 預防

### Q22（答 A）子宮頸 HSIL
- **核心知識**：基底層 2/3 以上 atypia = HSIL（CIN II-III）
- 對比：LSIL = 下 1/3 + koilocyte（HPV cytopathic effect）
- **進展風險**：HSIL → invasive Ca 約 30%

### Q23（答 B）蝕骨細胞活性異常
- **核心知識**：
  - ① Osteopetrosis = osteoclast **不能** resorb（mutation in CA II 等）
  - ③ Post-menopausal osteoporosis = 雌激素↓ → osteoclast **過度活化**
  - ⑤ Paget disease = osteoclast **異常巨大**過度活化 + osteoblast 失調
  - ② Senile osteoporosis = osteoblast 老化 → 屬 osteoblast 問題
  - ④ Osteomalacia = vitamin D 缺乏 → 骨基質礦化不足，與 osteoclast 無關

### Q24（答 D）皮膚培養出致病菌
- **核心知識**：Erythema migrans = Lyme disease，**Borrelia burgdorferi** 可在病灶處培養
- 其他 erythema 多為免疫反應性（無活菌）：
  - EM → drug、HSV 反應
  - Erythema induratum → 結核反應（panniculitis）
  - Erythema nodosum → 結核、Strep、藥物、IBD 反應

### Q25（答 A）感染性關節炎錯誤敘述
- **核心知識**：兩歲以下幼兒最常見化膿性關節炎 = **Staphylococcus aureus**
  - Salmonella 在**鐮形細胞病**患者特別好發（不是 default）
- 年齡分層：
  - 新生兒：S. aureus、GBS、Gram(-) bacilli
  - < 2 歲：S. aureus、H. influenzae（疫苗後減少）
  - 青少年：**Neisseria gonorrhoeae**
  - 成人：S. aureus

## Part 2 — 115 年第 1 次詳解

### Q26（答 B）Phosphatidylserine 外翻
- **核心知識**：細胞凋亡早期，**flippase 失活** → PS 從內層翻到外層 → "eat me" 信號 → 巨噬細胞吞噬
- **臨床應用**：Annexin V 染色（PS-binding）= 早期凋亡偵測
- **延伸**：對比細胞凋亡 vs 壞死的吞噬機制（凋亡無發炎；壞死有發炎）

### Q27（答 A）生理性肥大主要路徑
- **核心知識**：**PI3K/AKT/mTOR** = 生理性肥大（運動員心臟、孕婦子宮）
- 對比病理性肥大（壓力負荷高血壓心臟） = **GPCR 活化、Ca++/calcineurin/NFAT**
- 延伸：PIK3CA 突變 → 腫瘤性 PI3K 過度活化 → Alpelisib 為標靶

### Q28（答 A）病毒-疾病錯誤配對
- **正確**：Hairy leukoplakia = **EBV**（HIV 患者）
- HPV → 條狀疣、口腔 papilloma、SCC
- 其他三個都正確：HHV-8/PEL、EBV/NK-T、JCV/PML
- **教學重點**：免疫缺陷者「病毒-腫瘤」清單

### Q29（答 C）腎臟移植排斥分類
- **核心知識**：
  | 排斥類型 | 時程 | 主要病理 | 治療 |
  |---|---|---|---|
  | Hyperacute | 分鐘-小時 | 預存抗體 → 血栓 | 切除腎臟 |
  | Acute cellular | 天-週 | **淋巴細胞浸潤 tubulitis/vasculitis**，**C4d 陰性** | 高劑量類固醇、ATG |
  | Acute antibody-mediated | 天-週 | DSA、**C4d 陽性**、micropvasc 炎 | 血漿置換、IVIG、Rituximab |
  | Chronic | 月-年 | 間質纖維化、tubular atrophy、血管硬化 | 難治 |
- 本題：淋巴浸潤 + C4d 陰性 = **Acute cellular**

### Q30（答 C）Pellagra
- **核心知識**：Niacin (B3) 缺乏 → **3D**：
  - Dermatitis（光照部位粗糙、casal necklace）
  - Diarrhea
  - Dementia
- **常見族群**：玉米為主食地區（玉米中 niacin 為結合型不易吸收）、酗酒、Carcinoid syndrome（色胺酸被轉作 serotonin）

### Q31（答 A）MI 3 小時肉眼變化
- **核心知識**：MI 病理時間軸
  | 時間 | 肉眼 | 顯微 |
  |---|---|---|
  | 0-4h | **無變化** | 無變化（電顯可見） |
  | 4-12h | 黑暗色 | early coagulative necrosis、wavy fiber |
  | 12-24h | 大理石樣 | 完整 coagulative necrosis、early PMN |
  | 1-3d | 黃色軟化（中央） | PMN 浸潤 |
  | 3-7d | 中央黃褐、邊緣紅褐 | Macrophage 清除 |
  | 1-2w | 凹陷紅灰色 | 肉芽組織 |
  | > 2w | 灰白堅硬瘢痕 | 膠原沉積 |
- 3 小時 → 答案 (A) 看不出變化（顯微才能看到 early changes）

### Q32（答 D）CHF 水腫機轉
- **核心知識**：水腫 4 大機轉
  | 機轉 | 例子 |
  |---|---|
  | ↑ 靜水壓 | **CHF**、深部靜脈栓塞 |
  | ↓ 滲透壓（osmotic） | 肝硬化、腎病、營養不良 |
  | 淋巴阻塞 | 乳癌術後、絲蟲 |
  | ↑ 通透性 | 發炎、過敏 |
- CHF 主要機轉是**靜水壓**升高，不是滲透壓下降

### Q33（答 A）Giant cell arteritis
- **核心知識**：GCA = 老年人（**> 50 歲**，常 > 70）、**大中血管**、**頭部**為主（颞動脈、眼動脈）
- 病理：**granulomatous** + 多核巨細胞 + IEL 斷裂
- 併發症：**失明**（眼動脈分支阻塞，**急症**需立即類固醇）
- 對比：
  - Polyarteritis nodosa → 中血管、腎/腸繫膜
  - Takayasu → 大血管、**主動脈弓**、年輕女性
  - Kawasaki → 小孩、冠狀動脈瘤

### Q34（答 C）IDA 實驗值
- **核心知識**：IDA 五大實驗值
  | 項目 | 變化 |
  |---|---|
  | Hct、Hb | ↓ |
  | MCV | ↓ (microcytic) |
  | MCHC | ↓ (hypochromic) |
  | Serum iron | ↓ |
  | **TIBC** | **↑** |
  | Ferritin | ↓ |
  | 轉鐵蛋白飽和度 | ↓ |
- 對比 anemia of chronic disease：Ferritin ↑、TIBC ↓

### Q35（答 D）G6PD 缺乏症
- **核心知識**：**X-linked recessive**（男性多）
- 病生：缺 G6PD → NADPH ↓ → glutathione 不能還原 → 紅血球抗氧化能力↓
- 誘發：fava beans、antimalarials、sulfa 藥
- 鏡下：**Heinz body**、**bite cell**

### Q36（答 D）Celiac disease
- **核心知識**：Celiac disease 相關淋巴瘤 = **Enteropathy-associated T-cell lymphoma (EATL)**，非 B 細胞
- 其他重點：HLA-DQ2/DQ8、上皮內淋巴球↑、絨毛萎縮、隱窩增生
- **抗體**：Anti-tissue transglutaminase（tTG）、anti-endomysial
- **治療**：終身無麩質飲食

### Q37（答 A）HGSC 基因突變
- **核心知識**：High-grade serous ovarian Ca = **TP53 突變 > 96%**
- 衍生：BRCA1/2 mutation 是另一重要群（占 20%）
- 形態：psammoma body、SET pattern、輸卵管 fimbrial origin（STIC）
- **治療意義**：BRCA+ 適用 PARP inhibitor

### Q38（答 B）MSA α-synuclein
- **核心知識**：MSA = α-syn 包涵體位於 **oligodendrocyte**（GCI = glial cytoplasmic inclusion）
- **重要對比**：
  | 疾病 | α-syn 包涵體位置 |
  |---|---|
  | PD、DLB | **神經元**（Lewy body） |
  | **MSA** | **Oligodendrocyte**（GCI） |
- 臨床：MSA = parkinsonism + autonomic dysfunction + cerebellar ataxia 混合

### Q39（答 D）RA 與 TNF
- **核心知識**：**TNF-α** 是 RA pannus 形成的關鍵 cytokine
- 拮抗劑：Infliximab、Etanercept、Adalimumab
- 其他 RA 治療靶點：
  - IL-6 → Tocilizumab
  - CTLA4-Ig → Abatacept
  - CD20 → Rituximab
  - JAK → Tofacitinib

### Q40（答 D）Pulmonary HTN 錯誤敘述
- **核心知識**：**Plexiform lesion = 晚期** PAH 病理特徵（不是早期）
- 早期：medial hypertrophy、intimal proliferation
- 中期：concentric intimal fibrosis
- 晚期：**plexiform lesion**、necrotizing arteritis
- WHO 分類 PAH 5 群：(1) 特發/遺傳/藥物 (2) 左心病變 (3) 肺病/缺氧 (4) PE (5) 多因素

### Q41（答 B）大葉性肺炎分期
- **正確順序**：**congestion → red hepatization → gray hepatization → resolution**
  - Congestion：充血、漿液性滲出
  - Red hepatization：RBC + neutrophil + fibrin 滿肺泡
  - Gray hepatization：RBC 溶解、PMN 持續、fibrin 為主
  - Resolution：吞噬清除、結構恢復
- 題目把 gray 和 red 順序顛倒 = 錯
- **常見病原**：S. pneumoniae（大葉性）vs S. aureus/H. influenzae/Mycoplasma（支氣管性）

### Q42（答 D）支氣管擴張症症狀
- **核心知識**：支擴常見 = 慢性咳嗽 + 大量膿痰 + 咳血 + 杵狀指
- **過度換氣**（hyperventilation）是焦慮症狀，非典型支擴症狀
- 病因：CF、Kartagener、結核、ABPA、麻疹/百日咳後遺、IgG/IgA 缺乏

### Q43（答 D）自體免疫肝炎
- **核心知識**：自免肝炎血清 = **IgG ↑**（不是 IgM）
- IgM 升高 → primary biliary cholangitis (PBC)
- AIH 分型：
  - Type 1：ANA、ASMA → 成人女性、HLA-DR3/DR4
  - Type 2：LKM-1、LC1 → 兒童
- 治療：類固醇 ± azathioprine

### Q44（答 B）UC 錯誤敘述
- **核心知識**：UC = **僅累及結腸與直腸**（不包括 ileum）；發炎僅限**黏膜層**
- "backwash ileitis"（嚴重 UC 偶爾累及末端 ileum）≠ "涉及 ileum"
- **重要對比 UC vs Crohn**：
  | 特徵 | UC | Crohn |
  |---|---|---|
  | 位置 | 結腸 + 直腸 | 末端迴腸最常見，任何處 |
  | 連續性 | 連續 | skip lesion |
  | 深度 | mucosa | transmural |
  | 肉芽腫 | 無 | 有 (30%) |
  | 假息肉 | 常見 | 罕見 |
  | 廔管 | 罕見 | 常見 |
  | 結腸 Ca 風險 | ↑↑ | ↑ |
  | p-ANCA / ASCA | p-ANCA+ | ASCA+ |

### Q45（答 B）ACTH-independent Cushing
- **核心知識**：
  - ACTH-dependent (80%)：垂體腺瘤（Cushing disease）、ectopic ACTH（SCLC、carcinoid）
  - ACTH-independent (20%)：**腎上腺腺瘤** > 增生 > 腺癌
- **診斷流程**：24h cortisol → ACTH 測定 → MRI/CT
- 形態：腎上腺單側腫大（腺瘤）vs 雙側（hyperplasia）

### Q46（答 B）FSGS
- **核心知識**：原發性 FSGS 從 **juxtamedullary** 腎絲球先病變
- 兒童腎病症候群最常見 = **Minimal change disease**（不是 FSGS；成人 FSGS 較常見）
- 類固醇反應：FSGS **不佳**（Minimal change 反應極佳）
- EM：足細胞 foot process 廣泛融合（類似 MCD）+ 節段性硬化、玻璃樣變

### Q47（答 A）橋本氏甲狀腺炎
- **核心知識**：anti-TPO + anti-thyroglobulin（≥ 90% 患者陽性）
- **病理**：lymphocyte 浸潤、germinal center、**Hürthle cell**（eosinophilic 變性的甲狀腺濾泡細胞）
- 臨床：起初可能短暫 thyrotoxicosis → 後期 hypothyroidism
- 併發：MALT lymphoma 風險 ↑

### Q48（答 A）Papillary DCIS vs Papilloma
- **核心知識**：兩者都有 papilla，但 papilloma **保有肌上皮**，DCIS **無肌上皮**
- IHC：p63、SMA、calponin → 標記肌上皮
- 教學重點：「**有 myoepithelium = 良性**」是乳房病理鐵則

### Q49（答 A）侵犯性乳癌 ER/PR/HER2 組合
- **核心知識**：四大分子分型
  | 分型 | 比例 | 治療 |
  |---|---|---|
  | **Luminal A**：ER+/PR+/HER2−（Ki67 低） | **最多 ~50%** | 內分泌治療為主 |
  | Luminal B：ER+/PR+/HER2±（Ki67 高） | ~15-20% | 內分泌 + 化療 ± HER2 標靶 |
  | HER2-enriched：ER−/PR−/HER2+ | ~10-15% | Trastuzumab 等 HER2 標靶 |
  | Triple negative：ER−/PR−/HER2− | ~15-20% | 化療為主、近年免疫療法 |

### Q50（答 C）Duret hemorrhage
- **核心知識**：Duret hemorrhage = transtentorial（uncal）herniation → 下行性壓迫 brainstem → **midbrain/pons 中央線性出血**
- 其他疝出：
  - Subfalcine（cingulate）：扣帶迴在大腦鎌下，可能壓 ACA
  - Transtentorial（uncal）：颞葉鉤回，壓 CN III、PCA、brainstem（**Duret**）
  - Tonsillar：小腦扁桃體 → foramen magnum，壓延髓 → 呼吸停止
