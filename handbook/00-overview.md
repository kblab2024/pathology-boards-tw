## 二、核心結論（TL;DR）

### 2.1 近 2 次考題 vs 過去 10 年的差異
| 面向 | 變化 |
|------|------|
| 系統分布 | 五大主力（腫瘤、CV、Resp、肝膽、血液）**未變**，仍佔 ~60% |
| 免疫/自體免疫題比例 | **明顯上升**（114-2 + 115-1 達 12%） |
| 腎臟泌尿題 | 微升（含移植病理） |
| 題目深度 | **臨床整合題大幅增加**（40% vs 歷史 15-20%） |
| 分子病理 | 從邊緣考點變成**必考**（TP53、HLA、HNF1α、C4d 等） |
| 形態描述題 | 減少，被分子分型題取代 |

### 2.2 三個質變方向
1. **形態 → 分子分型**（HCA、卵巢癌、肺癌、乳癌）
2. **單一疾病 → 機轉鏈**（疾病 → 分子 → 治療標靶）
3. **常見 → 易混淆配對**（病毒-腫瘤、年齡分層病原體、各種 erythema）

### 2.3 最該補強的 14 個新冒出/少見考點
1. Acute cellular vs antibody-mediated rejection (C4d)
2. 病毒-腫瘤罕見配對（HHV-8、JCV、HPV/EBV）
3. MSA α-synuclein in oligodendrocyte
4. HNF1α-inactivated HCA
5. Pellagra（Niacin 缺乏 + 3D）
6. Erythema migrans / Lyme
7. Renal oncocytoma EM 粒線體
8. Xanthogranulomatous pyelonephritis + Proteus
9. FSGS 起源 juxtamedullary
10. ACTH-independent Cushing → adrenal adenoma
11. EATL（T-cell！Celiac）
12. Plexiform lesion = 晚期 PAH
13. Duret hemorrhage / Transtentorial herniation
14. HLA-DQ2/8 (Celiac)、DR3/DR4/DRB1 (AIH)

## 三、教學流程建議

### 短程衝刺（4-6 週）
適合考前 1-2 個月：
```
Week 1：模擬考（章節 02、03）→ 找出弱項系統
Week 2-3：依弱項挑系統 + 配對題卡（章節 04）+ buzzword（章節 08）
Week 4：14 張弱點補強卡（章節 09）
Week 5：分子-疾病-治療速查（章節 01）+ IHC（章節 07）
Week 6：再次模擬考 + 錯題回顧
```

### 中程系統（3 個月）
```
Month 1：12 週課程的前 4 週（W1-W4）
Month 2：W5-W8
Month 3：W9-W12 + 模擬考 + 衝刺
```

### 長程訓練（6-12 個月）
```
Month 1-3：基礎病理（細胞、發炎、免疫、腫瘤總論）
Month 4-6：各系統病理（依 W4-W11 排程）
Month 7-9：分子病理深化（IHC、基因突變、靶向治療）
Month 10-12：弱點補強 + 大量題庫練習 + 模擬考
```

## 五、教學成效驗收（第 12 週）

五項驗收標準：
1. **冷門考點 mini test**：從 14 卡各出 1 題，學生需答對 ≥ 11/14
2. **分子鏈條口試**：隨機抽 5 個分子，講出「基因→蛋白→疾病→治療」
3. **整套模擬考**：50 題目標 ≥ 70%（70 分）
4. **易混淆配對快測**：30 對中隨機 10 對，60 秒鑑別
5. **錯題回顧**：學生提交個人錯題本（系統 + 主題 + 錯誤類型分類）

**達標**：5 項中 ≥ 4 項達成。

## 六、教學重點優先序口訣

> **「五大穩、四小升、一新趨」**
>
> - **五大穩**：腫瘤、CV、Resp、肝膽、血液 — 仍是基本盤
> - **四小升**：免疫、腎泌（含移植）、神經、感染（含病毒-腫瘤）— 比重微升
> - **一新趨**：分子病理 + 機轉鏈 + 臨床整合 — 不會就考不過
