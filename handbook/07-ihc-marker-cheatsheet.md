## 經典「組合判讀」題

### 例 1：肺結節，IHC：**TTF-1+、Napsin A+、CK7+、p40−、ALK+**
**解讀**：肺腺癌 + ALK rearrangement → **Crizotinib / Alectinib**

### 例 2：頸部淋巴結，IHC：**CD20+、CD5+、Cyclin D1+、CD23−**
**解讀**：Mantle cell lymphoma → **Ibrutinib / Venetoclax**

### 例 3：腎上腺切除，IHC：**Synaptophysin+、Chromogranin+、Inhibin−、S100+ (sustentacular)**
**解讀**：Pheochromocytoma（S100+ sustentacular cell 為特色）

### 例 4：乳房切片，IHC：**ER 95%、PR 80%、HER2 1+（FISH−）、Ki67 8%、E-cadherin 失**
**解讀**：Invasive lobular carcinoma、Luminal A 亞型

### 例 5：腹膜切片，IHC：**Calretinin+、WT1+、D2-40+、CK5/6+、MOC-31−**
**解讀**：Mesothelioma（非腺癌）

## 必背口訣

1. **Hodgkin RS = CD15+CD30+，CD45−**（"15-30 過半 not 45"）
2. **CLL = 5+23**（CD5+、CD23+；雙正面）
3. **MCL = 5+11;14**（CD5+、Cyclin D1+，t(11;14)）
4. **Mesothelioma = CWD**（**C**alretinin、**W**T1、**D**2-40）
5. **HCC = HAG**（**H**epPar-1、**A**rginase-1、**G**lypican-3）
6. **HGSC = TP53 + WT1 + p16 + PAX8**
7. **GIST = KIT + DOG1**（不是 fibroma 不是 leiomyoma）
8. **Ewing = CD99 membrane + FLI1**（"99 是 Ewing"）
9. **Rhabdomyosarcoma = Myogenin + MyoD1**（"我的肉是動的"）
10. **Renal IHC = 7 多就是色素 / 透明 = chromophobe / clear cell**
