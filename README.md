## 快速開始

### 一鍵 render（需 Python 3.11+）

```powershell
py -3.11 -m venv .venv
.venv\Scripts\activate
pip install -r tools/render/requirements.txt
python -m playwright install chromium
python tools/render/render.py --all
```

完成後 `rendered/` 會出現 11 個 PDF。

### 其他指令

```powershell
python tools/render/render.py --file 03    # 只 render 第 3 章
python tools/render/render.py --master     # 只 render 合訂本
python tools/render/render.py --clean      # 清空 rendered/*.pdf
```

或在 Claude Code 內：

```
/render-all
```

## 渲染管線（技術簡介）

```
handbook/*.md ─┐
               ├─→ markdown-it-py + 引用剝除 ─┐
handbook/*.csv ┘                              │
                                              ├─→ Jinja2 模板組裝
封面 / colophon / 章節分隔 / TOC / 封底  ─────┤
                                              ├─→ Playwright (headless Chromium)
CSS (補習班配色、字型、表格、@page、headers) ─┘
                                              ├─→ 每章 PDF (含 orientation)
                                              ├─→ pypdf 合併
                                              └─→ 合訂本 PDF
```

**核心設計約束**：
- ✅ **AI 透明度**：封面、版權頁、頁腳、所有 handbook 原始檔頂部都明確標示 Claude Code 署名與「對照標準教科書」警語。
- ❌ **零原始檔路徑引用**：渲染後內文不出現「foo.md / bar.json / [L42]」這類引用。Audit 由 `markdown_to_html.audit_path_references` 在 render 前 assertion；漏掉就會 raise `RuntimeError`。
- 🈲 **保留國考代碼**：`104-1`、`115-1-87`、`104-114` 等是國考題目編號，視為合法內容不剝除。
